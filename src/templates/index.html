Welcome to Tracebin!

Tracebin is a small server application that enables you to post and display Python tracebacks for later analysis.

Posting a traceback to Tracebin is done with the help of offlinetb. Here is a small code snippet to get you started: 

import json
import urllib2
import offlinetb
...
try:
   raise Exception("This is what you'd like to report")
except Exception:
   response = urllib2.urlopen("{{url}}", json.dumps(offlinetb.distill()))

The server returns JSON data containing, among else, the URL to the traceback that you just posted: 

traceback_url = json.loads(response.read())['url']
